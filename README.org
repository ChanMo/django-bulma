** Quick Start

Install Package

#+BEGIN_SRC bash
  pip install django-bulmacss
#+END_SRC

修改`settings.py`

#+BEGIN_SRC bash
  INSTALLED_APPS = [
      'bulma',
      ...
  ]

  FORM_RENDERER = 'django.forms.renderers.TemplatesSetting'
#+END_SRC
